課題の自動配信 デモ

毎朝の定時配信・自動採点・正答率に応じたレベル調整まで、AIワークフローで半自動化。 回答は Googleスプレッドシートに集計し、解説も自動添付できます。

AIエージェント 自動化 適応学習
schedule_send

定時配信

問題を毎朝○時に自動送信。休校日や長期休暇も制御可能。

assignment_turned_in

自動採点・解説

選択式は自動採点、記述式もAI補助。解説・復習リンクを自動生成。

trending_up

レベル調整

正答率・解答時間に応じて難易度を自動調整し、最適化。

1

配信チャネルを選ぶ（Forms / Classroom / メール等）

Googleフォーム＋スプレッドシート、Google Classroom、LMS、またはメール配信など、学校環境にあわせて選択できます。

2

AIで問題を自動生成し、解説も用意

目標コンテンツ（語彙・計算・長文・考察など）をAIに渡し、出題・模範解答・解説をテンプレに沿って生成します。

初期プロンプト例（コピペOK）

「一次方程式の文章題（中1）」の小問を毎日3問生成。
形式：Q（問題文）/ Choices（最大4つ）/ Answer / Explanation（誤答のよくある考え方を1つ含む）/ Difficulty（★1-5）
制約：数値は毎回変える。用語は教科書準拠。冗長な背景は省く。
3

定時配信 → 回答収集 → スプレッドシート集計

スケジューラで配信を自動化し、回答はスプレッドシートへ自動蓄積。採点・解説返信も自動化できます。

スプレッドシート列例

  • timestamp / student_id / class
  • q1_answer / q1_correct / q1_time
  • score_total / level_next

採点ロジック例

  • 選択式：一致=1点
  • 記述式：正規化→類似度閾値
  • 誤答タイプを分類し解説テンプレ差し込み

レベル調整例

  • 正答率≧80% → 難度+1 / 出題数+0
  • 50–79% → 難度±0 / 出題数+1（復習）
  • ＜50% → 難度-1 / ヒント付与
4

個別フィードバックと復習リンクを自動返信

生徒の誤答傾向に応じて、よくある誤りの会話例や、次に取り組むとよい練習問題へのリンクを自動で提示します。

返信テンプレ（Markdown）

## 今日のふり返り（{日付}）
- 総合：{score_total}点 / 正答率 {accuracy}%
- よくある誤り：{misconception_summary}
- 会話例：{misconception_dialogue_snippet}
- 次にやると良い：{next_exercises}（リンク）

運用上の注意（大事）

  • 評価の公平性：AI採点は補助。最終判断は教員が確認。
  • データ保護：生徒IDの扱い・共有範囲・保存期間を明確化。
  • 著作権：引用問題は出典明記。配布条件を遵守。
